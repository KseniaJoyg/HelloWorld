Заголовок виджета.

Содержимое виджета. Для того, чтобы изменить тему оформления щелкните на элементе в верхнем левом углу экрана.

Обратите внимание: из-за особенностей работы нашего редактора код данного предмета не будет работать корректно. Для того, чтобы опробовать виджет themeswitcher создайте HTML файл на Вашем компьютере и скопируйте в него код из примера.
